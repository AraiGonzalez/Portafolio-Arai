Experiencia Laboral

DevelopFit

  • Actualmente estoy desarrollando un sistema de gestión para un bazar. Se utiliza Angular para el frontEnd y Java para el backEnd con Spring Boot como framework, junto con JPA y Hibernate para la persistencia, y MySQL como gestor de base de datos. Las APIs necesarias que se crearon están basadas en el modelo MVC. Se implementó una arquitectura multicapa, que incluye Controller, Model, Repository, Service, Annotations e inyección de dependencias. Además, se implementó autenticación con JWT y seguridad en las rutas, con gestión de roles.
  • Desarrollé el fronEnd para un sistema de gestión de inventario para tienda en linea. Se utilizo angular: Se utilizaron herramientas como la creación de componentes, templates, servicios, módulos, formularios reactivos, sesión de usuario, uso de Bootstrap, directivas, databinding, interfaces, routing, lazy load, guards, pipes.
